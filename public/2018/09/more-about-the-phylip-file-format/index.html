More about the PHYLIP file format

September 21, 2018 in study_notes, perl

How to restore the truncated sequence identifiers.

In the previous post, I mentioned that the length the sequence identifiers in the strict PHYLIP file format cannot exceed 10 characters. Longer sequence names will be truncated:

# strict.phy
     3    15
homo_sapie GATAATGCTG ACTAC
ailuropoda GATAATGCTG ACTAT
felis_catu GATAATGCTG ACTAT

We can probably still figure out that they are human üë∂, panda üêº and cat üê±. But it might get confusing when the data contains many many sequences.
I wrote a perl script to restore the sequence names and change the alignment files to relaxed PHYLIP format.
We will need a text file containing the full species names:

# species_list.txt
homo_sapiens
ailuropoda_melanoleuca
felis_catus

The code:

Usage: ./change_phy_species_names.pl strict.phy species_list.txt
And the output will be:

# relaxed.phy
     3    15
homo_sapiens                 GATAATGCTG ACTAC
ailuropoda_melanoleuca       GATAATGCTG ACTAT
felis_catus                  GATAATGCTG ACTAT
